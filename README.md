# Academic Skills

This is a skill directory I started maintaining in my first pre-doctoral year. It is expected to be maintained and updated over the long term.

These skills are not demo artifacts. They are used in real research and learning work every day, and I hope they can also help more researchers build readable, auditable, and humane academic workflows.

| Category | Skill | Description |
| --- | --- | --- |
| Research computing and analysis governance | `stepwise-r-project` | Maintains readable R analysis projects with standard folders, Memory logs, RStudio-style script structure, and function-level R Markdown audits. |
| Literature learning and academic writing | `adhd-academic-tutor` | Supports source-facing literature reading, academic writing pattern learning, time calibration, and long-term learning memory. |
| Execution support and task decomposition | `adhd-tasker` | Breaks broad or overwhelming tasks into concrete micro-steps with generous time boxes and low-pressure feedback loops. |
